Tauri in Action

Building StandClock

What is Tauri? ü¶Ä

A modern toolkit for building desktop apps

  • Frontend: Use any web framework (React, Vue, Solid.js, etc.)
  • Backend: Powered by Rust (fast, safe, efficient)
  • Uses your OS: Leverages system's built-in browser engine
  • Tiny bundles: Much smaller than bundling a whole browser
  • Cross-platform: Write once, run on Windows, Mac, Linux

Think of it as: Web UI + Rust power + System browser = Small, fast desktop app

The Problem: Sitting Too Much

  • Back pain, poor posture, health issues from sitting all day
  • When you're focused on work, it's easy to forget to take breaks
  • Sitting 8+ hours daily increases health risks by 30%
  • We need gentle reminders to stand up and move

Why a Desktop App?

  • Always there: Runs in the background, you don't need to remember to open it
  • Smart notifications: Pops up when it's time to take a break
  • No phone needed: Your phone might be in another room
  • Where you work: Most office work happens at a desk
  • Your data stays private: Everything saved on your computer, not the cloud

What's Wrong with Existing Apps?

  • Website timers: Easy to accidentally close the browser tab
  • Phone apps: Your phone might be far away, drains battery
  • Heavy apps: Slow to start, eat up computer memory
  • Browser extensions: Only work when browser is open

What we need: A small, fast app that just works

The Big Decision: Electron vs Tauri

Two ways to build desktop apps using web technology (HTML, CSS, JavaScript).

But they work very differently under the hood.

Electron vs Tauri: The Numbers

Think: Tauri is like a compact car, Electron is like an SUV

Why Electron is Popular

  • Lots of help available: Many tutorials and examples online
  • Easy to learn: Just use JavaScript, no new language
  • Big apps use it: VS Code, Slack, Discord, Figma
  • Tons of ready-made tools: Easy to add features
  • Works the same everywhere: Looks identical on Windows, Mac, Linux

Why I Chose Tauri for StandClock

  • Super lightweight: Won't slow down your computer
  • Runs all day: Uses less memory, perfect for background apps
  • Fast to start: Opens instantly when you boot your computer
  • Safer: Rust prevents many common security bugs
  • Small updates: Download only what changed, not the whole app
  • Future ready: Will work on phones too (coming soon!)

The Downsides of Tauri

  • Rust is harder to learn than JavaScript
  • Smaller community - fewer examples online
  • Newer technology - less mature tools
  • Some quirks on different operating systems

My decision: For an app that runs all day, being small and fast is worth the extra learning!

What StandClock Does

  • Track Your Progress: See how many sessions you completed

How It's Built

  • User Interface: Solid.js (like React, but faster and smaller)
  • Core Engine: Rust + Tauri 2.0
  • Data Storage: SQLite (keeps your history on your computer)
  • Looks: Tailwind CSS + DaisyUI (for those 32 themes!)
  • Build Tool: Bun (super fast JavaScript toolkit)

Simple version: Web technology on the outside, Rust power on the inside

Works on All Computers

  • Windows: Uses Edge's browser engine
  • Mac: Uses Safari's browser engine
  • Linux: Uses WebKit

Each one uses your computer's built-in browser tech - that's why it's so small!

What Went Great! ‚úÖ

  • Super light: People say it doesn't slow down their computer at all
  • Fast to build: Solid.js made development quick
  • Easy to organize: Clean code structure makes changes easy
  • Ready-made tools: Tauri's database plugin just worked
  • Caught bugs early: TypeScript + Rust found errors before users did
  • 32 beautiful themes: Got them almost for free with DaisyUI!

My First Attempt Failed üí•

Original plan: Go full Rust!

  • Idea: Use Rust + WASM for frontend with Sycamore framework
  • Goal: Pure Rust everywhere - no JavaScript at all

What went wrong:

  • Very few docs: Sycamore was new, hard to find examples
  • Had to write everything: No ready-made components
  • Took forever: Simple things became complicated
  • Hard to maintain: Would be nightmare to update later

Smart move: Switched to Solid.js - better docs, easier development, still fast!

What Was Hard üòÖ

  • Learning Rust: Took me way longer than expected
  • Finding solutions: Not as many tutorials as Electron
  • Debugging: Rust errors can be confusing
  • Platform differences: Had to tweak things for Windows/Mac/Linux
  • New technology: Some Tauri features weren't well documented yet

Should YOU Use Tauri?

Use Tauri if:

  • Your app needs to run in the background without slowing things down
  • You want a small download size
  • You're starting a new project (easier than converting old one)
  • You're willing to learn Rust (or already know it!)
  • Speed and efficiency matter more than quick development

Stick with Electron if:

  • Your team only knows JavaScript and needs to move fast
  • You need lots of ready-made plugins and tools
  • You don't have time to learn Rust
  • Your app doesn't need to run 24/7
  • You need the app to look exactly the same everywhere

Bottom line: Both are great! Pick based on your needs.

Key Takeaways üí°

  • Tauri makes tiny, fast desktop apps - perfect for tools that run all day
  • The speed difference is real - but you have to learn Rust
  • Modern web tools work great - Solid.js, TypeScript, Tailwind
  • Good code structure matters - makes everything easier later
  • Choose wisely: Match the tool to your project's needs
